A complete documentation is available here: [https://slidesk.github.io/slidesk](https://slidesk.github.io/slidesk/)

A good example of a presentation using the features: [SliDesk Talk](https://github.com/slidesk/talk-slidesk)

Write your talk/presentation in Markdown, generate it and visualize it in Web.

SliDesk is a new talk engine like RevealJS developped with [Bun](https://bun.sh).

Plugins & Components can be found here : https://github.com/slidesk/slidesk-extras or in the HUB : https://slidesk.link

A VSCode extension is available too : https://github.com/slidesk/vscode-sdf-language
