# Create a deployment file

```sh
slidesk deploy -t link mytalk
```

You have to specify the target wanted:

```sh
  -t, --target <param>      generate a deploy file for 'github', 'gitlab' or 'link' (slidesk.link)  (default: ) 
```

This command allows you to create deployment files for `GitLab`, `GitHub` or `slidesk.link`

More information on [Deploy Section](/deploy)
